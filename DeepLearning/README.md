### 稀疏表示

在信号与图像处理过程中，模型是至关重要的。
借助合适的模型，可以处理各种任务，如：去噪，恢复，分离，内插，外插，压缩，采样，分析和合成、检测、识别等。
该模型的核心在于线性代数中研究的一个简单的欠定线性方程组。
